Puppet Class: chkrootkit

Defined in:
manifests/init.pp

Overview

This class configures chkrootkit

Examples:

Standard configuration with Hiera

include '::chkrootkit'
chkrootkit::diff_mode: true
chkrootkit::run_daily: true:
chkrootkit::run_daily_opts: '-q'

Standard configuration directly setting variables

class { 'chkrootkit':
  diff_mode         => true,
  run_daily         => true,
  run_daily_opts    => '-q',
}
