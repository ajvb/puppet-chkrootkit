Puppet Class: chkrootkit

Defined in:
manifests/init.pp

Overview

This class configures chkrootkit

Examples:

Standard configuration with Hiera

include '::chkrootkit'
chkrootkit::diff_mode: true
chkrootkit::config:
  RUN_DAILY: 'true'
  RUN_DAILY_OPTS: '-q'

Standard configuration directly setting variables

class { 'chkrootkit':
  diff_mode     => true,
  config        => {
    'RUN_DAILY'         => 'true',
    'RUN_DAILY_OPTS'    => '-q',
  }
}
